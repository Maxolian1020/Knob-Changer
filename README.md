# Volume Redirector

Volume Redirector is a Windows tray app that intercepts the system media volume keys and redirects them to a selected playback device.

## Features

- Global media-key interception for volume up, volume down, and mute
- Select a target playback device from the tray menu
- Persistent device selection across app restarts
- Lightweight Windows-style volume overlay
- Startup with Windows support

## Build

From the project directory:

```powershell
dotnet build
```

## Run

```powershell
dotnet run
```

## Notes

This app targets .NET 8 on Windows.
